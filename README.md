# Vue3 Big File Upload Solution

大文件上传解决方案演示项目。

## 目录结构

- `client/`: 前端项目 (Vue 3 + TypeScript + Vite)
- `server/`: 后端服务 (Koa + TypeScript)

## 快速开始

### 前端 (Client)

```bash
cd client
npm install
npm run dev
```

### 后端 (Server)

```bash
cd server
npm install
npm run dev
```

## 技术栈

- **Frontend**: Vue 3, TypeScript, Vite
- **Backend**: Koa, TypeScript, koa-body (multipart support)
